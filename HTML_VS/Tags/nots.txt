**block 
<div>  -->  defines a section(container) in a ducument.its a block element(takes max width and line breaks) 
        --> used to add CSS styles.
        --> allows multiple <div> with different inline css(style attribute.) 
        --> side by side (use css float:left/right  property.)  OR "display:inline" property.

<span>     --> defines a section in a document. used to add a style in a block element(p,h,div)
            ex: <h2> this is an <span  style="font-size:120px;" > holy book  </span> </h2> 

<iframe>    --> used to display a web page inside a web page. 
            --> src, title, name,width,height (attributes). 
            ex: <iframe src="example.html" title="demo" width="200" height="150" name="iframe1" > 
            --> opening a link in a iframe. 
                 ex: <iframe src="url of  web page" name="name1" > 
                     <a  href="url of link" target="name1" > click the link </a> 
                     -->target attribute where to open link.
