   Block & inline Elements 

   Block Elements --> <div> <p> <pre> <h1>...<h6> <ol><ul><li><dt>  <table>  <form> <header>  <footer>
       ** block lements takes a line break atbegining and at ending.

       ** bolck_level elements can be define  inside  other block level element  only not in inline- elements.
       
    Inline-elements 
       --> <span> {<b><i><u><sub><sup><s><mark><small>} <a> <img> <input>  <label> <select>
       ** inline elements does not take line break.
